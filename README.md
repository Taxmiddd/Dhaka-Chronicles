# Dhaka Chronicles

![Dhaka Chronicles Logo](./public/DC%20(2).svg)

> **Delivering timely, accurate and engaging news content that resonates.** Reporting from the heart of Dhaka with grit, style, and absolute truth.

Dhaka Chronicles is a premium, high-performance news platform designed for the digital generation. Built with a neo-brutalist aesthetic, it combines cutting-edge web technologies with a bold, uncompromising editorial voice.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Database**: [Supabase](https://supabase.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Runtime**: Optimized for **Edge Runtime** (Cloudflare Pages)

---

## ✨ Key Features

- **🚀 Global Reach**: Fully optimized for the Edge, ensuring lightning-fast load times globally via Cloudflare.
- **📰 Dynamic Content**: Real-time news delivery powered by Supabase.
- **🎨 Neo-Brutalist Design**: A unique, high-contrast visual identity that stands out in the digital landscape.
- **🛡️ Private Studio**: A secure, password-protected administrative hub for content management.
- **🔍 Advanced Search**: Instant article discovery through a dedicated search interface.
- **📱 Responsive by Design**: A seamless experience across mobile, tablet, and desktop.

---

## 🏗️ Architecture

The project follows a modern monorepo-style structure:

- `/src/app`: Next.js App Router for high-performance routing.
- `/src/components`: Atomic design system with reusable UI components.
- `/src/lib`: Centralized data fetching, authentication logic, and utilities.
- `/supabase`: Database schema and migration scripts.
